# Girls Who Code - Calvin University College Loop Website

Welcome to the official website repository for the Girls Who Code College Loop at Calvin University! This website serves as a hub for our community, providing information about our mission, events, and how to get involved.

## About Us

Girls Who Code College Loop at Calvin University is dedicated to closing the gender gap in technology. We provide a supportive community where women can learn, grow, and excel in computer science and related fields.

**Our Mission:** Empowering women in technology through community, coding, and collaboration.

## Features

-   **Responsive Design:** The website is fully responsive and looks great on all devices, from desktops to mobile phones.
-   **Smooth Navigation:** Includes smooth scrolling for a seamless user experience.
-   **Interactive Elements:** Features hover effects, mobile navigation menu, and dynamic content.
-   **Animations:** subtle fade-in animations on scroll to enhance visual appeal.
-   **Contact Form:** A functional-looking contact form (currently client-side demonstration).

## Technologies Used

-   **HTML5:** For semantic structure.
-   **CSS3:** For styling, including Flexbox and Grid layouts, and custom animations.
-   **JavaScript:** For interactivity, mobile menu toggling, and scroll effects.

## Getting Started

To view the website locally:

1.  Clone this repository or download the files.
2.  Open the `index.html` file in your web browser.
